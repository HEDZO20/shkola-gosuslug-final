Школа Госуслуг — финальная версия учебной платформы

Что есть в проекте:
- Главная страница с редактируемыми текстами и блоками.
- Курс с поэтапным открытием уроков.
- Отдельная страница урока: видео, текст, шаги, ошибки, практика, тест.
- Личный кабинет ученика с прогрессом и уведомлениями.
- Курс после завершения всех уроков.
- Админ-панель: уроки, черновики, предпросмотр, файлы, ученики, статистика, настройки сайта.
- Настройки сайта без кода: название, логотип, тексты, кнопки, WhatsApp, цвета, ширина сайта.

Первый запуск:
1. Создайте проект Supabase.
2. Откройте Supabase → SQL Editor → New query.
3. Скопируйте весь код из setup.sql и нажмите Run.
4. В Supabase откройте Project Settings → API.
5. Скопируйте Project URL и anon public key / publishable key.
6. Вставьте их в supabase-config.js.
7. Зарегистрируйтесь на сайте.
8. В Supabase → Table Editor → profiles поставьте для своего email role = admin.
9. Откройте Личный кабинет — админ автоматически попадет в панель управления.

Если вы уже запускали старый setup.sql:
- Запустите новый setup.sql повторно. Он добавит новые поля настроек и не должен создавать дубликаты уроков.

Важно:
- Не вставляйте service_role key в сайт.
- Для сайта нужен только anon public key / publishable key.
