ШКОЛА ГОСУСЛУГ — стабильная и ускоренная версия

Что добавлено:
1. PWA-кэш: сайт быстрее открывается на телефоне и может показать сохраненную версию при слабом интернете.
2. Быстрая загрузка кабинета: кабинет пробует получить профиль, уроки, прогресс и материалы одним RPC-запросом get_my_dashboard().
3. Быстрая загрузка админки: админка пробует получить настройки, уроки, материалы, учеников, прогресс, активность и ошибки одним RPC-запросом get_admin_dashboard().
4. Индексы Supabase для ускорения таблиц profiles, lessons, quiz_questions, lesson_progress, materials, site_events.
5. Журнал ошибок site_errors: ошибки загрузки сайта записываются в Supabase и показываются в админке → Проблемы учеников → Ошибки сайта.
6. Видео теперь грузятся легче: preload="metadata", iframe — loading="lazy".
7. На телефоне облегчены blur/анимации, добавлены офлайн/онлайн уведомления.
8. Иконка ГОС и manifest сохранены для iPhone “На экран Домой”.

Как установить:
1. Распакуйте архив.
2. Загрузите файлы на GitHub поверх старых.
3. Не затирайте supabase-config.js, если там уже стоят рабочие Project URL и public key.
4. В Supabase откройте SQL Editor → New query.
5. Выполните файл OPTIMIZE_SUPABASE.sql.
   Можно также выполнить полный setup.sql — он ничего не удаляет, но OPTIMIZE_SUPABASE.sql короче и быстрее.
6. Откройте сайт с параметром ?v=20, чтобы браузер взял новую версию:
   https://ваш-сайт.github.io/ваш-репозиторий/?v=20

Важно:
- Service Worker может держать старые файлы в кэше. После обновления откройте сайт 1–2 раза или используйте ?v=20.
- Supabase на бесплатном тарифе после восстановления может первый раз отвечать медленнее. После первого открытия обычно становится быстрее.
