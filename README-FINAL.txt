Школа Госуслуг — финальная версия

Добавлено:
- восстановление пароля через email;
- поддержка подтверждения email;
- SEO-теги, иконка сайта и картинка для отправки ссылки;
- админ-раздел «Инструкция»;
- админ-раздел «Резерв»: экспорт/импорт уроков, тестов и настроек;
- подтверждения для опасных действий;
- улучшенный раздел файлов с удалением и проверкой размера;
- подсказки по формату видео;
- простая аналитика в админке: просмотры страниц, загрузки, активность;
- таблица site_events в setup.sql.

ВАЖНО:
Если сайт уже загружен, не затирайте supabase-config.js пустым файлом. Перенесите туда свои Project URL и anon/public key.

Если обновляете базу Supabase, выполните свежий setup.sql. Он добавит таблицу site_events и новые политики.


ОБНОВЛЕНИЕ: назначение админов и выход на телефоне
1. Запустите новый setup.sql в Supabase, чтобы исправить защиту role/approval_status.
2. Чтобы дать админку, выполните: select * from public.make_user_admin('email@example.com');
3. На телефоне теперь есть нижняя панель с кнопкой «Выйти».
