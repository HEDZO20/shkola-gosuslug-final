Одноразовая настройка

После этой настройки уроки, видео, тексты, тесты, ученики и прогресс будут управляться через админку без редактирования кода и без GitHub.

  1. Создайте проект в Supabase.
  2. Откройте Supabase → SQL Editor → New query.
  3. Откройте файл setup.sql, скопируйте весь текст и нажмите Run.
  4. Supabase → Project Settings → API → скопируйте Project URL и anon public key.
  5. Откройте файл supabase-config.js и вставьте эти 2 значения.
  6. Зарегистрируйтесь на сайте.
  7. В Supabase → Table Editor → profiles → у вашего email поставьте role = admin.
  8. Откройте Личный кабинет: если у email роль admin, откроется панель управления.

Открыть сайт Открыть личный кабинет

Не вставляйте service_role key в сайт. Нужен только anon public key.
